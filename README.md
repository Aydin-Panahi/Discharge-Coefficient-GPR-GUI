# Side Weir Discharge Coefficient Predictor

**Unified GPR Model for Sharp- and Broad-Crested Trapezoidal Side Weirs**

A user-friendly MATLAB GUI for rapid prediction of discharge coefficient (Cd) using the Gaussian Process Regression model developed in our study.

## Features
- Supports both sharp-crested and broad-crested side weirs
- Instant prediction with one click
- Schematic diagrams for both weir types
- Input range validation

## How to Use
1. Download and run `GPR_SideWeir_GUI_MODERN.exe`
2. Select weir type (Sharp or Broad)
3. Enter parameters: y₁, L, P, t, z, Fr₁
4. Click **Predict Cd**
